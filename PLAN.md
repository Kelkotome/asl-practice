# ASL Practice App — MVP

Web app that helps ASL learners improve signing through real-time camera-based feedback, powered by RAG from ASL-LEX data.

**User flow:** Pick a sign → watch reference video → attempt on camera → get AI coaching feedback.

**Deployment:** `deafened.org/practice` (reverse proxy to Vercel-hosted Next.js app).
